# Memory Fragmentation Simulation

A C program that simulates how memory allocation works and how fragmentation can occur over time.

The program demonstrates basic operating system memory management concepts through a simple simulation.

---

## Features

- Simulates memory block allocation
- Demonstrates internal and external fragmentation
- Displays memory usage and allocation status
